# Supply-Chain-Analytics-Dashboard
📌 Project Overview

This project presents an interactive Supply Chain Analytics Dashboard built using Power BI to analyze inventory and product distribution across different supply chain dimensions.

The dashboard provides insights into product availability, supplier distribution, customer demographics, shipping routes, and transportation modes. It helps businesses monitor inventory levels, understand product demand, and optimize logistics decisions.

By transforming raw supply chain data into meaningful visualizations, the dashboard enables data-driven decision making for inventory management and logistics optimization.
